Mantenimiento Correctivo

Diagnóstico y solución de fallos comunes en ordenadores y portátiles.

💀 Pantallazo azul (BSOD)

Generalmente causado por controladores, memoria o temperatura.

  1. Anota el código de error (ej. 0x0000007B).
  2. Prueba a iniciar en modo seguro.
  3. Verifica la memoria con MemTest86.
  4. Actualiza o reinstala controladores.

🔥 Sobrecalentamiento y apagados repentinos

El equipo se apaga solo para proteger componentes.

  • Limpia ventiladores y disipadores (ver preventivo).
  • Reemplaza la pasta térmica.
  • Usa una base refrigeradora para portátiles.
  • Comprueba que el ventilador funcione.

🐢 Lentitud extrema

Puede deberse a disco lleno, malware o falta de RAM.

  • Revisa el Administrador de tareas (CPU, RAM, disco).
  • Escanea con Malwarebytes.
  • Si usas HDD, cambia a SSD (mejora enorme).
  • Aumenta la memoria RAM si es posible.

🔌 El equipo no enciende

Sin signos de vida o luces parpadeantes.

  1. Prueba otro cargador / toma de corriente.
  2. Retira la batería (si es extraíble) y prueba solo con adaptador.
  3. Resetea la memoria CMOS (desconecta pila unos minutos).
  4. Si nada funciona, posible placa base dañada.
